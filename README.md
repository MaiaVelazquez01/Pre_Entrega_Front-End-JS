# 🧪 FarmaLab

FarmaLab es una página web tipo landing desarrollada con HTML y CSS que simula una farmacia online. Su objetivo es presentar productos farmacéuticos, mostrar reseñas de clientes y ofrecer un formulario de contacto funcional.

## 🎯 Propósito del proyecto

El propósito de esta página es practicar y demostrar habilidades en desarrollo web front-end, incluyendo:

- Diseño responsive (adaptado a dispositivos móviles y desktop)
- Uso de Flexbox y Grid
- Estructuración semántica en HTML
- Estilado con CSS
- Implementación de formularios funcionales con Formspree

## 🧩 Secciones principales

- **Inicio:** Banner principal de la página.
- **Productos:** Catálogo de medicamentos con descripción y precios.
- **Reseñas:** Opiniones de clientes.
- **Contacto:** Formulario para envío de mensajes.

## 📱 Responsive Design

La página está diseñada con un enfoque responsive mediante media queries, adaptando la estructura y los elementos a diferentes tamaños de pantalla para garantizar una correcta visualización en dispositivos móviles, tablets y desktop.

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3
- Google Fonts
- Font Awesome
- Formspree (para envío de formularios)
